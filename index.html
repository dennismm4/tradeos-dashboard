🔑 Binance API Setup
COMPLETE
✓
Create Binance account
You already have a Binance account — this step is done.
✓
Generate API keys
Go to Binance → Profile → API Management → Create API. Enable Spot Trading only. Disable withdrawals for safety.
3
Create .env file
In the same folder as binance_bot.py, create a file called .env and add:
BINANCE_API_KEY=your_key_here
BINANCE_API_SECRET=your_secret_here
4
Install dependencies
pip install python-binance python-dotenv pandas
5
Run in paper mode first (2 weeks minimum)
python binance_bot.py --mode paper
Watch the log file. Only go live after consistent paper profits.
6
Go live
python binance_bot.py --mode live
📊 Exness MT5 Setup
IN PROGRESS
1
Open Exness account (Kenya)
Visit exness.com → Register → Choose Standard account. Deposit $30 via M-Pesa. Takes ~10 minutes.
2
Download MetaTrader 5
Download MT5 from the Exness dashboard. Log in with your Exness account credentials.
3
Install Python MT5 library
pip install MetaTrader5
4
Add Exness credentials to .env
MT5_LOGIN=your_account_number
MT5_PASSWORD=your_password
MT5_SERVER=Exness-MT5Real
5
Run Exness bot in demo mode
python exness_bot.py --mode demo
🚀 Deploy to GitHub Pages
✓
Download the project files
You have: dashboard.html, binance_bot.py, README.md
2
Create GitHub repository
Go to github.com → New repository → Name it tradeos-dashboard → Public.
3
Upload dashboard.html
Drag and drop dashboard.html into the repo. Rename it to index.html for GitHub Pages.
4
Enable GitHub Pages
Settings → Pages → Source: main branch → Save. Your dashboard will be live at username.github.io/tradeos-dashboard
